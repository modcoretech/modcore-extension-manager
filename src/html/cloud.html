Sync Options

Configure automatic syncing, what data to sync, and retention policies.

Selective Sync

Choose which data categories to sync. Unchecking a category prevents it from being sent to or restored from the cloud.

Automatic Syncing

Periodically saves your selected data to the cloud in the background.

Sets how often the automatic background sync runs. Chrome's alarm API is designed for approximate intervals - actual sync times may vary based on browser activity and system resources.

Retention Policies

Control how many history versions and log entries are kept. Older entries are trimmed automatically. Hard limits apply regardless of policy to protect storage quota.

Controls how many cloud sync versions are retained. Hard maximum: 25 versions.

Controls how many sync log entries are kept. Hard maximum: 500 entries.
