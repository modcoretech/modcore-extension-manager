# Security Policy for modcore Extension Manager

At **modcore**, the security and privacy of our users are our top priorities. If you discover a security issue in **modcore Extension Manager**, we encourage you to report it responsibly so we can fix it quickly.

---

## Reporting a Security Issue

Please report vulnerabilities **privately** via email:

**Email:** [ngde@web.de](mailto:ngde@web.de)  

When reporting, include as much of the following as possible:  

- **Description of the issue** - what is happening and why it’s a problem.  
- **Steps to reproduce** - if applicable, a simple way for us to see the issue.  
- **Version info** - which version of the extension you tested.  
- **Potential impact** - what could happen if the issue is exploited.  
- **Proof of concept or code snippets** - only if necessary to demonstrate the problem.  

> Do **not** disclose the vulnerability publicly until it has been fixed.

---

## Our Commitment

Once a security issue is reported:  

1. We will acknowledge your report within **48 hours**.  
2. Our team will review the issue and prioritize fixing it based on severity.  
3. We will develop, test, and release a fix as quickly as possible.  
4. After the fix is deployed, we may provide a **public advisory** describing the issue and resolution (without sensitive details).  

We take every report seriously and strive to maintain a secure and private experience for all users.

---

## Tips for Users

While we work to address issues:  

- **Keep your extension updated** - always use the latest version.  
- **Be cautious with third-party extensions** - only use trusted sources.  
- **Review extension permissions** - make sure they match the expected functionality.
