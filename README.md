# modcore Extension Manager
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fmodcoretech%2Fmodcore-extension-manager.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fmodcoretech%2Fmodcore-extension-manager?ref=badge_shield)


**modcore Extension Manager** is a **privacy-focused**, **open-source** browser extension that gives users full control over their installed extensions. Developed and maintained by [modcore](https://github.com/modcoretech), it provides a comprehensive toolset to manage, analyze, and organize browser extensions **securely**, without collecting or transmitting personal data.  

Designed for privacy-conscious users and developers, modcore Extension Manager ensures that all operations are performed locally on your device, offering complete transparency and control over your browser environment.

---

## Overview

modcore Extension Manager is a lightweight yet powerful extension management tool. It allows users to:

- View all installed extensions and their detailed metadata.  
- Inspect requested permissions and potential security risks.  
- Enable, disable, uninstall, or organize extensions efficiently.  
- Maintain multiple profiles and extension groups for different use cases.  

Unlike many extension management tools, **modcore EM does not collect user data**. Your browsing activity and extension configurations remain entirely on your device.  

This tool is ideal for:

- **Privacy-conscious users** who want full control over their browser.  
- **Developers** managing multiple extensions or testing configurations.  
- **Power users** who want insight into browser performance and security.  

---

## Key Features

modcore Extension Manager offers a rich set of features for comprehensive extension control:

### **Extension Management**
- Enable, disable, or uninstall extensions with a single click.  
- Group extensions into custom profiles for work, gaming, or other use cases.  
- Quickly search and filter installed extensions by name, type, or status.  

### **Detailed Insights**
- Inspect permissions requested by each extension.  
- Identify potential risks, conflicts, or deprecated extensions.  
- View Version information, ID information, and much more.  

### **Privacy and Security**
- Zero data collection — all processing occurs locally on your device.  
- No telemetry, tracking, or analytics integrated in the extension.  
- Strict Content Security Policy (CSP) prevents malicious code execution.  

### **Offline Support**
- Fully functional without an internet connection, except for GitHub API metadata retrieval.  
- Cached data is stored locally and does not contain personally identifiable information.  

### **Backup and Restore**
- Export extension settings, groups, and profiles to a JSON file for backup or migration.  
- Import previous configurations easily to maintain your preferred setup.  

### **GitHub Integration**
- Fetch public metadata for extensions available on GitHub.  
- Ensure accurate and up-to-date information without transmitting your data.

## Installation

You can install modcore Extension Manager manually in your Chromium-based browser:

1. Go to the [Releases page](https://github.com/modcoretech/modcore-extension-manager/releases).  
2. Download the latest `.zip` release.  
3. Open your browser’s **Extensions** page (`chrome://extensions/`).  
4. Enable **Developer Mode**.  
5. Click **Load unpacked** and select the folder extracted from the zip file.  

Once installed, the **modcore** icon will appear in your browser toolbar.

---

## Getting Started

After installation:

1. Click the **modcore** icon in your browser toolbar.  
2. Browse and manage your installed extensions.  
3. Create profiles or groups to organize your extensions.  
4. Use the backup feature to save your settings.  

All operations are performed locally — no sign-in or cloud account is required.
